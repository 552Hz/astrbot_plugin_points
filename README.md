# 积分管理系统插件

一个功能完善的群聊积分管理系统插件，支持账户注册、积分查询、管理员操作、排行榜、转账等功能。

## 功能特性

### 首次使用
- **注册账户** - 用户自定义用户名，账号自动绑定QQ号

### 用户功能
- 查询自己的积分和群排名（生成精美卡片图片）
- 查看群积分排行榜
- 向其他用户转账积分
- 修改用户名
- 查看帮助信息

### 管理员功能
- 给指定用户增加积分
- 扣除指定用户的积分
- 直接设置用户积分数值
- 重置用户积分为初始值

### 积分卡片预览
当用户使用 `/积分` 指令时，机器人会回复一张精美的卡片图片，显示：
- 用户名
- 账号（QQ号）
- 当前积分
- 群排名信息

## 支持平台

- QQ 官方机器人 (qq_official)
- OneBot v11 (aiocqhttp)
- KOOK
- Discord
- Telegram

## 安装方法

### 方法一：从源码安装

1. 将整个 `astrbot_plugin_points` 文件夹复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或在 WebUI 中重载插件
3. 在插件管理页面启用该插件

### 方法二：从 GitHub 安装

在 AstrBot WebUI 的插件市场页面搜索"积分管理"进行安装。


## 重要提示

**所有用户必须先注册账户才能使用积分功能！**

未注册用户使用任何积分相关指令都会收到提示，要求先注册。

## 指令说明

### 首次使用

| 指令 | 说明 |
|------|------|
| `/注册 用户名` | 注册账户（用户名2-20字符，不能重复） |

### 用户指令

| 指令 | 别名 | 说明 |
|------|------|------|
| `/积分` | `/points`, `/score` | 查询自己的积分和群排名 |
| `/排行榜` | `/rank`, `/ranking` | 查看群积分排行榜 |
| `/转账 @用户 数值` | - | 向指定用户转账积分 |
| `/修改用户名 新名字` | - | 修改您的用户名 |
| `/积分帮助` | `/points_help` | 查看帮助信息 |

### 管理员指令

| 指令 | 说明 |
|------|------|
| `/加积分 @用户 数值` | 给指定用户增加积分 |
| `/扣积分 @用户 数值` | 扣除指定用户的积分 |
| `/设置积分 @用户 数值` | 直接设置用户的积分为指定值 |
| `/重置积分 @用户` | 重置指定用户的积分为初始值 |

### 使用示例

**注册账户：**

## 配置说明

插件支持在 WebUI 中可视化配置以下参数：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 新用户初始积分 | 100 | 注册时获得的初始积分数量 |
| 最低积分限制 | 0 | 用户积分可以扣除到的最低值 |
| 单次操作最大积分 | 10000 | 单次增加或扣除积分的最大值 |
| 排行榜显示人数 | 10 | 排行榜指令显示的前N名用户 |

## 数据存储

插件使用两个数据文件：

1. **accounts.json** - 用户账户数据
```json
{
  "123456789": {
    "username": "小明",
    "registered": true
  }
}
```

2. **points.json** - 积分数据
```json
{
  "群ID_QQ号": 积分数值
}
```

**注意**：插件更新或重装不会丢失数据，因为数据存储在 AstrBot 的数据目录下。

## 开发说明

### 核心类

- `AccountData`: 用户账户管理类，负责账户的注册、查询、用户名修改
- `PointsData`: 积分数据管理类，负责积分的增删改查
- `PointsPlugin`: 插件主类，继承自 `Star`

### 关键方法

**账户相关：**
- `is_registered()`: 检查用户是否已注册
- `register()`: 注册新账户
- `update_username()`: 修改用户名

**积分相关：**
- `get_points()`: 获取用户积分
- `set_points()`: 设置用户积分
- `add_points()`: 添加/扣除积分
- `get_group_ranking()`: 获取群排行榜
- `get_user_info()`: 获取用户信息（含排名）

### 权限控制

管理员指令使用 `@filter.permission_type(filter.PermissionType.ADMIN)` 装饰器进行权限控制，只有群管理员才能使用。

## 常见问题

**Q: 为什么我无法使用积分功能？**
A: 您需要先使用 `/注册 用户名` 注册账户后才能使用积分功能。

**Q: 用户名可以重复吗？**
A: 不可以，用户名在系统中是唯一的，不能与已有用户名重复。

**Q: 排行榜显示的是用户名还是QQ号？**
A: 已注册用户显示用户名，未注册用户显示"用户QQ号"。

**Q: 积分数据存储在哪里？**
A: 数据存储在 `data/plugin_data/astrbot_plugin_points/` 目录下，包含 accounts.json 和 points.json 两个文件。

**Q: 管理员也需要注册吗？**
A: 是的，所有用户（包括管理员）在使用积分相关指令前都需要先注册。
